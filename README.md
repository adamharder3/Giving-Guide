# Giving Guide

## Overview
A web app that helps users discover charities based on their interests and values.

### Tech Stack
- **Frontend:** React
- **Backend:** Node.js + Express
- **Database:** SQLite

### Setup Instructions

#### 1. Clone and install
```bash
git clone https://github.com/adamharder3/Giving-Guide.git
cd Giving-Guide
```

#### 2. Set up backend
```bash
cd backend
cp .env.example .env
npm install
npm run dev
```

#### 2. Set up frontend
```bash
cd ../frontend
cp .env.example .env
npm install
npm start
```
